# Security Policy

## Versiones soportadas

Actualmente solo se da soporte a la última versión estable de SpotiFLAC-ES:

- v1.0.0-es (release actual): soportada

Versiones anteriores o builds modificados por terceros pueden no recibir soporte.

## Cómo reportar una vulnerabilidad

Si detectas un posible problema de seguridad en SpotiFLAC-ES:

- No abras un issue público con detalles técnicos.
- Envía un correo a: drextenmax@gmail.com
- Incluye:
  - Versión de SpotiFLAC-ES
  - Sistema operativo
  - Pasos para reproducir el problema
  - Cualquier log o captura relevante (sin datos sensibles)

Intentaré revisar y responder en un plazo razonable. No hay programa de recompensas/bounty, pero agradezco mucho los reportes responsables.

## Alcance

Esta política aplica únicamente al fork **SpotiFLAC-ES** publicado en este repositorio.  
Para vulnerabilidades en el proyecto original SpotiFLAC, repórtalas directamente en el repositorio de afkarxyz.
